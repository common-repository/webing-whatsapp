Start WhatsApp Chat Directly From Your Website.

== Description ==

This plugin let you to begin a WhatsApp Chat Directly From Your Website.

click on the button to activate WhatsApp chat app and write your message to your contact!

The plugin detects if the user uses mobile to show a different url address to open the WhatsApp on Android or IOS systems,
and it also detects if the user uses desktop to open WhatsApp Web.

1. Display a green WhatsApp Icon on the bottom of the page
1. Optional number edit on the backend
1. Optional automatic message to be sent on the backend
1. support and service for operating the plugin
1. The code is written according to the regulations and requirements of WordPress

== Installation ==

Install the plugin by:

1. Upload `webing-whatsapp` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to `Settings -> WhatsApp Button` page and setup you phone number and message.

== Frequently Asked Questions ==

= What happens when i click on the WhatsApp icon? =

a. if its on computer - the WhatsApp Web opens on your browser.
b. if its on mobile - the WhatsApp App opens on your cellphone.

= Can I use the prefixes of all the countries in the world? =

Yes.
